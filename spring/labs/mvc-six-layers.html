이번 실습의 목표

Spring MVC 의 6 계층은 「고급 식당의 분업 구조」와 정확히 매칭된다.

1
코드 작성
@Controller     // 종업원
class HomeController {
    @Autowired Service service;       // 셰프 호출
}
@Service        // 메인 셰프
class HomeService {
    @Autowired Mapper mapper;         // 창고 관리자 호출
}
@Repository     // 창고 관리자
interface HomeMapper { ... }
