1
코드 작성
항목         쿠키                  세션
저장위치     브라우저              서버 메모리
보안         약함                  강함
용량         ~4KB                  큼
용도         장바구니, 다크모드    로그인 상태, 권한

// 쿠키 — 다크모드 설정
Cookie c = new Cookie("theme", "dark");

// 세션 — 로그인 사용자
session.setAttribute("loginUser", member);
