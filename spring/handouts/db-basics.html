DB 기초

Spring 기초
SMHRD

학습 목표

  • DB 기초 의 핵심 개념을 안다
  • DB = 엑셀 시트 의 동작을 안다
  • DB 가 익숙한 엑셀 시트로 보인다.

1. 문제

DB 가 추상적이라 어렵게 느껴진다.

2. 새 도구

DB = 엑셀 시트: 데이터베이스는 익숙한 엑셀 시트와 똑같이 생겼다. 시트 = 테이블, 행 = 레코드, 열 = 컬럼.

3. 코드

-- 테이블 만들기 (CREATE TABLE) CREATE TABLE mymember ( id VARCHAR(50) PRIMARY KEY, pwd VARCHAR(20) NOT NULL ); -- 행 추가 INSERT INTO mymember (id, pwd) VALUES ('hong', 'temp1234');

4. 정리

  • 테이블 = 엑셀 시트 한 장
  • 행(Row) = 데이터 한 건
  • 열(Column) = 속성
  • PK = 고유 식별자
  • FK = 다른 테이블 참조

5. Before / After

전 차시

DB 가 추상적이라 어렵게 느껴진다.

이번 차시
