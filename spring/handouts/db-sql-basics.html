SQL 핵심 4종

Spring 기초
SMHRD

학습 목표

  • SQL 핵심 4종 의 핵심 개념을 안다
  • CRUD 4 종 의 동작을 안다
  • 4 가지 쿼리만으로 회원·게시판 기본 동작을 다 만들 수 있음을 안다.

1. 문제

SQL 책 한 권을 다 봐야 할 것 같지만 — 사실 4 가지가 80% 다.

2. 새 도구

CRUD 4 종: Create·Read·Update·Delete = INSERT·SELECT·UPDATE·DELETE. 이 네 가지로 게시판이 만들어진다.

3. 코드

-- READ (조회) SELECT id, pwd FROM mymember WHERE id = 'hong'; -- CREATE (입력) INSERT INTO mymember (id, pwd) VALUES ('hong', 'temp1234'); -- UPDATE (수정) UPDATE mymember SET pwd = 'newpw' WHERE id = 'hong'; -- DELETE (삭제) DELETE FROM mymember WHERE id = 'hong'; -- 게시판 (myboard) SELECT num, title, writer, content FROM myboard ORDER BY num DESC; INSERT INTO myboard (title, writer, content) VALUES ('첫 글', 'hong', '안녕');

4. 정리

  • SELECT — 조회 (가장 많이 씀)
  • INSERT — 추가
  • UPDATE — 수정
  • DELETE — 삭제
  • WHERE 가 핵심

5. Before / After

전 차시

SQL 책 한 권을 다 봐야 할 것 같지만 — 사실 4 가지가 80% 다.

이번 차시

4 가지 쿼리만으로 회원·게시판 기본 동작을 다 만들 수 있음을 안다.
