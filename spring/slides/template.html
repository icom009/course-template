차시 제목

Part · 비유

[강사 노트] 수업 시작 시 분위기 가이드. 비전공자가 부담 없이 들어올 수 있게.

학습 목표

  • 핵심용어 1이 무엇인지 비유로 설명할 수 있다
  • 핵심용어 2의 동작 원리를 그림으로 그릴 수 있다
  • 실제 화면에서 핵심용어 3를 직접 확인할 수 있다
학습 목표는 "할 수 있다" 형태로. 학생이 끝나고 나서 자기 검증할 수 있게.

⚠️ 어디가 아팠나

이전 버전의 문제

...구체적인 불편함을 학생이 느낄 수 있게 서술. 가능하면 직접 시연한 후 들어옴.

이 자리에서 "그래서 우리가 새 도구를 배우는 거예요"의 동기가 만들어진다. 첫 차시처럼 이전 버전이 없으면 "왜 이게 필요한가" (학생의 일상 호기심)로 대체.

🛠️ 새 도구

도구 이름 (English Name)

한 줄 설명 + 일상 비유.

[비유 다이어그램] ↓ ↓ [원리 설명]
새 개념은 반드시 일상 비유와 함께. 이론 먼저 말하지 말고 비유 먼저.

💻 코드로 만나기


// 여기에 실제 코드 예시
