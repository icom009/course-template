핵심 용어

CRUDCreate·Read·Update·Delete. 데이터의 4 가지 기본 동작.
WHERE조회·수정·삭제 시 조건 지정.

1. 문제

전 차시까지의 답답함

SQL 책 한 권을 다 봐야 할 것 같지만 — 사실 4 가지가 80% 다.

2. 새 도구

CRUD 4 종

Create·Read·Update·Delete = INSERT·SELECT·UPDATE·DELETE. 이 네 가지로 게시판이 만들어진다.

3. 코드

-- READ (조회)
SELECT id, pwd FROM mymember WHERE id = 'hong';

-- CREATE (입력)
INSERT INTO mymember (id, pwd) VALUES ('hong', 'temp1234');

-- UPDATE (수정)
UPDATE mymember SET pwd = 'newpw' WHERE id = 'hong';

-- DELETE (삭제)
DELETE FROM mymember WHERE id = 'hong';

-- 게시판 (myboard) 도 같은 4 동작
SELECT num, title, writer, content FROM myboard ORDER BY num DESC;
INSERT INTO myboard (title, writer, content)
VALUES ('첫 글', 'hong', '안녕하세요');

4. 정리

  • SELECT — 조회 (가장 많이 씀)
  • INSERT — 추가
  • UPDATE — 수정
  • DELETE — 삭제
  • WHERE 가 핵심

5. Before / After

전 차시 끝

SQL 책 한 권을 다 봐야 할 것 같지만 — 사실 4 가지가 80% 다.
