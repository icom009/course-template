DB

핵심 용어

테이블 (Table)데이터를 저장하는 표. 엑셀 시트와 동일.
PK (Primary Key)각 행을 고유하게 식별하는 컬럼. 회원 id 등.
FK (Foreign Key)다른 테이블의 PK 를 참조하는 컬럼.

1. 문제

전 차시까지의 답답함

DB 가 추상적이라 어렵게 느껴진다.

2. 새 도구

DB = 엑셀 시트

데이터베이스는 익숙한 엑셀 시트와 똑같이 생겼다. 시트 = 테이블, 행 = 레코드, 열 = 컬럼.

3. 코드

-- 테이블 만들기 (CREATE TABLE) — v2 최소형
CREATE TABLE mymember (
    id   VARCHAR(50)  PRIMARY KEY,
    pwd  VARCHAR(20)  NOT NULL
);

-- 행 추가
INSERT INTO mymember (id, pwd)
VALUES ('hong', 'temp1234');

4. 정리

  • 테이블 = 엑셀 시트 한 장
  • 행(Row) = 데이터 한 건
  • 열(Column) = 속성
  • PK = 고유 식별자
  • FK = 다른 테이블 참조
