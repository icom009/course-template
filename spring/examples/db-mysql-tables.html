핵심 용어

AUTO_INCREMENT새 행 추가 시 PK 가 자동 증가.
PRIMARY KEY행을 유일하게 식별하는 컬럼.
ALTER TABLE기존 테이블 구조를 바꾸는 명령. 차시가 진행되며 컬럼이 자라난다.

1. 문제

전 차시까지의 답답함

DB 도구 설치가 어렵게 느껴진다.

2. 새 도구

MySQL 8 + Workbench

MySQL 서버 설치 후 Workbench 또는 DBeaver 같은 GUI 로 접속. 회원·게시판 테이블 DDL 실행.

3. 코드

-- 회원 테이블 (v2 최소형)
CREATE TABLE mymember (
    id   VARCHAR(50)  PRIMARY KEY,
    pwd  VARCHAR(20)  NOT NULL
);

-- 게시판 테이블 (v5 최소형)
CREATE TABLE myboard (
    num     INT          PRIMARY KEY AUTO_INCREMENT,
    title   VARCHAR(100) NOT NULL,
    writer  VARCHAR(50)  NOT NULL,
    content TEXT         NOT NULL
);

👉 처음에는 일부러 최소 컬럼으로 시작합니다. 작성 시각·조회수·첨부 같은 컬럼은 후속 차시에서 ALTER TABLE 로 추가합니다.

4. 정리

  • mymember — 로그인 ID 와 비밀번호 두 칸
  • myboard — 글번호 / 제목 / 작성자 / 본문 네 칸
  • 차시가 진행되며 두 테이블에 ALTER 가 차곡차곡 쌓인다
